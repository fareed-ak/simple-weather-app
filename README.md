A simple weather app built using **HTML, CSS, and JavaScript** to practice working with real APIs and async JavaScript.

The app uses the **OpenWeatherMap API** to fetch current weather data.

---

## Features

- Search weather by **city name**
- Fetches real-time weather data
- Displays that data to the user

---

## How It Works

1. User enters a city name
2. City name is converted into coordinates using the **Geocoding API**
3. Coordinates are used to fetch weather data from the **Weather API**
4. Data is processed and displayed on the UI

---

## APIs Used

- OpenWeatherMap Geocoding API  
- OpenWeatherMap Current Weather API  

---

## Notes

- Made for learning purposes
- API key is entered manually (no backend)
- UI kept simple and minimal
